## Introduction

Aitrium makes it simple for physician scientists around the world to share standardized data with research partners.

For research organizations sourcing data, Aitrium offers a web application that allows you to:

- Create and manage Trials  
- Set de-identification and dataset requirement policies  
- Configure a data transfer integration  
- View dataset contributions  
- Manage participation  

For organizations contributing data to research projects, Aitrium offers AitriumOS for Windows and Mac that supports the de-identification, standardization, and transfer of medical imagery datasets (DICOM) and supplementary files to Trials.
